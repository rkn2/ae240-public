## Updating This Repository from Canvas

### 1. Export from Canvas
1. Go to Canvas Course Settings.
2. Click **"Export Course Content"** → **"Course"** → **"Create Export"**.
3. Download the `.imscc` file.

### 2. Move the Export
Rename the file to `export.imscc` and place it in the repo root.

### 3. Run the Update Script
```bash
python3 update_course.py
```

### 4. Commit
```bash
git add .
git commit -m "Update course content from Canvas export"
git push
```
