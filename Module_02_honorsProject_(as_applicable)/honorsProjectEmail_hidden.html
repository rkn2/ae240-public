Hi all! I'm writing to share a set of resources I've prepared to help you get started on your Honors Data Science Project. The baseline topic for this project is the "Tale of Two Cities" analysis of tornado damage, and these guides are designed to support that.

To guide you through each stage, I have created documents that cover the key concepts and provide sample Python code.

The first colab notebook is called  projectSummary and it is a high level and then weeklyPythonSamples gives you details about what you could do each week like:

  1. Week 1 - Intro to Data Wrangling: Covers the basics of loading, inspecting, and cleaning data (what we have been doing in class).
  2. Week 2 - Exploratory Data Analysis (EDA): Shows you how to use visualizations to find patterns and relationships in your data.
  3. Weeks 3 & 4 - Predictive Modeling: Walks you through the process of building and evaluating your first machine learning model.

You can find all of these documents on our Canvas page under the "HonorsProject" module, which is located near the top.

Important note!! The sample code in these guides uses a fictitious dataset about wind turbine performance, not the tornado data. This is intentional. The goal is for you to learn the methods and techniques in a practice environment. You will then need to think about how to apply these same techniques to the unique features of the Nashville and Mayfield tornado datasets for your project.

Please start by working through the "Week 1" guide. Review the materials and try running the code.

Finally, please keep in mind that the tornado dataset serves as a baseline for the Honors Project. My goal is for you to work on something that genuinely interests you. If you have access to a different dataset you're passionate about and have an idea for a project, please come talk to me! We can look at the data together and design an alternative project that fits your interests.

Let me know if you have any initial questions as you get started!
